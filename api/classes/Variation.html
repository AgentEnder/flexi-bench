FromCliArgs()​

static FromCliArgs(args): Variation[]

Parameters​

• args: (string | string[])[]

An array of options to pass to the CLI. If an element is an array, they will be treated as alternatives.

Returns​

Variation[]

An array of variations. Can be applied with withVariations.

Example​

// Creates 2 variations, both with --flag, but one with --no-daemon and the other with --daemon.
const variations = Variation.FromCliArgs([
 '--flag',
['--no-daemon', '--daemon'],
]);

// Creates 4 variations, with combinations of --a and --b.
const variations = Variation.FromCliArgs([
['--a', '--b'],
]);

FromContexts()​

static FromContexts<T>(key, values): Variation[]

Creates variations with context values for a single key. Useful for swapping implementations or configurations.

Type Parameters​

• T

Parameters​

• key: string

The context key to set

• values: readonly readonly [string, T][]

Array of [name, value] tuples

Returns​

Variation[]

An array of variations with context set

Example​

const variations = Variation.FromContext('processor', [
  ['loop', loopProcessor],
  ['reduce', reduceProcessor],
]);

benchmark.withVariations(variations);
benchmark.withAction((variation) => {
  const processor = variation.get<DataProcessor>('processor');
  processor.process(data);
});
